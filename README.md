# Mohan Kholiya — Portfolio Website

Personal portfolio site for Mohan Kholiya, Procurement & Supply Chain Leader with 16+ years across energy, oil & gas, utilities, and telecom.

Built with [Astro](https://astro.build) and Tailwind CSS.

## Sections

- **Hero** — positioning statement and key stats
- **Case Studies** — anonymised client engagements across energy, utilities, and telecom
- **Competencies** — core capability areas
- **Credentials** — education, certifications, and career footprint
- **Contact** — email and LinkedIn

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- [Astro 6](https://astro.build)
- [Tailwind CSS 4](https://tailwindcss.com)
- TypeScript
